Date Object Methods


Get Method Description
getDate() Returns the date of a Date object
getFullYear() Returns the year of a Date object
getMonth() Returns the month of a Date object
getDay() Returns the day of a Date object
getHours Returns the hour of a Date object
getMinutes() Returns the minutes of a Date object
getSeconds() Returns the seconds of a Date object
getMilliseconds() Returns the milliseconds of a Date object
getTime() Returns the time of a Date object


Set Method Description
setDate(date) Sets the date(1-31) of a Date object
setFullYear(year[, month, day]) Sets the year(4 digits) of a Date object
setMonth(month[, date]) Sets the month(0-11). You can include what is in the brackets for the date
setHours(hours[, minutes, seconds, milliseconds]) Sets the hours(0-23) of a Date object. You can include what is in the brackets for minutes, seconds, milliseconds
setMinutes(minutes[, seconds, milliseconds]) Sets the minutes (0-59) of a Date object. You can include what is in the brackets for seconds, milliseconds
setSeconds(seconds[, milliseconds]) Sets the seconds(0-59) of a Date object. You can include what is in the brackets for milliseconds
setMilliseconds(milliseconds) Sets the milliseconds(0-999) of a Date object
setTime() Sets the time as the number of milliseconds that have elapsed since 12:00am, January 1, 1970


Other Methods Description
toLocaleString() Converts a Date object to a string, set to the current timezone.
toString() Converts a Date object to a string
valueOf() Converts a Date object to a millisecond format
now() Returns the current time as the number of milliseconds that have elapsed since 12:00am, January 1, 1970
